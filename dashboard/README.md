# Dashboard Module

This directory contains the dashboard component of the Slack Code project. The dashboard provides a web interface for managing the Slack bot and viewing its interactions and analytics.

## Contents

- `index.html`: The main HTML file for the dashboard interface.
- `styles.css`: The CSS file for styling the dashboard.
- `scripts.js`: JavaScript file containing logic for user interactions and dynamic content updates.
- `data/`: Directory containing JSON files used for dynamic data rendering in the dashboard.
- `img/`: Directory for storing image assets used in the dashboard interface.
